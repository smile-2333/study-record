文法
1. 名1(人) は 名2(事) が 一类形/二类型 です
2. 名1(人) は 名2(事) が わかります／できます
3.  名 や 名
4. から 和 だかり（原因，理由）
5. 表示频率的副词
6. どうしてですか

1. 名1(人) は 名2(事) が 一类形/二类型 です
例:
小野さんは歌が好きです
王さんは肉が嫌いです
林さんは犬が怖い

2. 名1(人) は 名2(事) が 分かります／できます
例:
スミスさんは韓国語が分かります
私はコンピュータができません
小野さんは料理上手です

3.  名 や 名
例:
吉田さんは時々中国や韓国へ行きます
森さんはサツカー野球が好きます
私はお店で野菜や果物など買いました
　
4. から 和 だかり（原因，理由）
例:
森さんはお酒好きですから、每日飲みます。
寒いですから、窓を閉めます。
明日は休みです。だから、子供と動物園へいきます

5. 表示频率的副词
例:
私はたまにコンサートへ行きます。
李さんは時々映画を見ます
田中さんはあまり横浜へはいません

6. どうしてですか
例:
僕は来月またここい行きますよ。
どうしてですか。
ここで友達の結婚式がありますから。

文法
1. 名1　は　名2　より　一类形/二类形　です。
eg：
李さん　は　森田さん　より　若い　です。
中国　は　日本　より　広いです。
上海　は　北京 より　にぎやか　です。

2. 名1　より　名2　のほうが　一类形/二类形　です。
eg:
日本　より　中国　のほうが　広いです。
北京　より　東京　のほうが　暖かいです。
大阪　より　東京　のほうが　にぎやがです。

3. 名1　は　名2　ほど（一类形くないです）／（二类形ではありません）。
eg:
東京の冬　は　北京（の冬）　ほど　寒くないです。
神戸　は　大阪　ほど　にぎやかではありません。
私の部屋は王さんの部屋　ほど　きりいではありません。

4. 名1の中で　名2　が　一番　形容词　です。
eg:
スポーツの中で　サツカー　が　一番　おもしろいです。
日本料理の中寿司が一番おいしです。

5. 名1　と　名2　と　どちらが　形容词ですか。
eg:
日本語と英語と　どちらが　難しいですか。
--日本語のほうが難しいです。

コーヒーと紅茶と　どちらが　好きですか。
--紅茶のほうが好きです。
--どちらも好きです。

6. どの名（物）／いつ（时候）／どれ（物）／だれ（人）／何が／どこ　いちばん adj ですか。
eg:
この中で、どの料理が　いちばん　おいしい　ですか。
--この料理が　いちばん　おいしい　です。
季節の中で、いつがいちばん好きですか。
--私は春がいちばん好きです。

文法
1. 一类形て　一/二类形（并列）
ホテルの部屋広くて　明るい　です。
この本は安くて　おもしろい　です。
このりんごは　大きくて　おいしい　です。

2. 二类形で　一/二类形（并列）
森さんは親切で優しい　です。
この公園静かで広いです。

3. 名1　で　名2（并列）
スミスさんは旅行会社の社長で、営業部の部長です。
この機械は日中電気の製品で、中国製です。

4. 动　ています（结果状态）
わたしは北京に住ん　ていもす。
車を持つていますか。
--いいえ、持っていません。
田中さんを知っていますか。
--いいえ、知りません。

5. 小句　が　小句（转折）
3か月パンコンを勉强しましたが、まだあまりできません。
このレストランはおいしいですが、ちょっと高いです。

6. もだ　+ 动（否定）
何年ぐらい　日本語を　勉强しましたか。
--二年勉强しました、でも、まだ、あまりできません。
小野さんはもう結婚して　いますか。
--いいえ、まだ結婚していません。

文法
1. 名 が　欲しい　です
わたしは　新しい　洋服　が　欲しいです。
（あなた）何　が　欲しいですか。
--新しい　パソコン　が　欲しい　です。

2. 名 を　动　たい　です
（わたしは）映画　を　見たいです。
今日　は　お酒　を　飲みたくないです。
（あなたは）何　を　したいですか。
--何もしたくありません。　

3. 动　ませんか
いっしょに　お茶　を　飲みませんか。
少し　休みませんか。
明日 故宫へ　行きませんか。

4. 动　ましょう
ちょっと　休みましょう。
そろそろ　いきましょう。
いっしょに　お茶を　飲みましょう。

5. 疑问词 + です（不论什么）
李さん　何　を　食べたいですか。
--何でも　いいです。
いつでも　電話　を　して　ください。
だれ　でも　分かりもす。

6. ね（缓和语气）
日本料理で　何が　いちばん　好きですか。
--そうですね。やっぱり　お寿司ですね。
